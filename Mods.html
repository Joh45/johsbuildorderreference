Interactive Build Order Guide

Interactive Build Order Guide is a series of interactive tutorials for introducing players to some of the most important build orders for the game. The 12 scenarios included in this mod each feature step-by-step instructions in both text and audio, with voicework by ZeroEmpires. They also feature a scoring system that marks the player at several benchmarks, leading to a final grade (A+ to F). See the FAQ page for more details.
