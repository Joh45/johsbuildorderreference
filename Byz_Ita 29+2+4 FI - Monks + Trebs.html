Byzantines/Italians 29+2+4 FI - Monks + Trebs

A significantly faster version of the build which Byzantines and Italians can pull off, and a good strategy especially for flank position in team games. Below build is optimised for the scenario requirements which is very gold-heavy, but you will want more on wood/farms if adding pikes for example.
