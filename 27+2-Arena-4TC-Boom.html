27+2 Arena 4TC Boom

Only use when no possibility of needing to defend - this is an important build for pocket position in Arena teamgames. This build has a later Imperial time than a 3TC boom, but is needed for the more expensive unit compositions. Some versions of the build involve getting heavy plow when you hit castle which slightly delays your boom, but is worth considering if you are going for a food-heavy unit composition.
